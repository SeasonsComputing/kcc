Parameters:
- host host name for server

- notify URL. Will send an HTTP GET to {URL}?action=(startup|shutdown)&host={host}&port={port}
queue - queue size of incoming socket listeners
sendWaitSec - timeout for send (secs)
recvWaitSec - timeout for receive (secs)

true if initialized successfully
