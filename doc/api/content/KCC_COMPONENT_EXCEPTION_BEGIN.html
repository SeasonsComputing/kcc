Utility macro to create component exceptions.

Documentation

Utility macro to create component exceptions. Required to force some compilers to emit exception code for stateless/methodless type-only exceptions

Note: All component exceptions MUST derive from kcc::Exception

Stock Usage: KCC_COMPONENT_EXCEPTION(MyCompException)

Custom Usage: KCC_COMPONENT_EXCEPTION_BEGIN(MyCustCompException) String target; String regex; MyCustCompException(const String& what, const String& _target, const String& _regex) : Exception(what), target(_target), regex(_regex) {} KCC_COMPONENT_EXCEPTION_END
