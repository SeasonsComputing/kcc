Documentation

Component factory. All component modules must have only 1 factory. Factory's should be specialized to add additional construction param's or utility methods or to create other components from this module.

virtual IComponent* construct() = 0
Factory method to construct a component. All component implementations must have at least an empty constructor for creation. Ownership of component is consumed by caller.
